Tecnología completa, sin complejidad innecesaria

Stack del producto URLoft pensado para shipping rápido: backend robusto, datos embebidos y ecosistema listo para web, extensión e integraciones IA.

Core tecnico

  • Backend: Bun + TypeScript.
  • Data: SQLite + FTS5 para bús**queda full-text.
  • Auth: Better Auth con sesiones stateful.
  • Workers: procesos asíncronos para tareas pesadas.

Ecosistema de producto

  • Sitio actual: esta home esta renderizada con Bun + EJS.
  • Canales: extensión Chrome + PWA instalable.
  • IA: MCP Server y Web Skill para search/lookup/extract.

Menos complejidad operativa, más velocidad de producto.
