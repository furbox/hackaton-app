Tecnologia completa, sin complejidad innecesaria

Stack del producto URLoft pensado para shipping rapido: backend robusto, datos embebidos y ecosistema listo para web, extension e integraciones IA.

Core tecnico

  • Backend: Bun + TypeScript.
  • Data: SQLite + FTS5 para busqueda full-text.
  • Auth: Better Auth con sesiones stateful.
  • Workers: procesos asincronos para tareas pesadas.

Ecosistema de producto

  • Sitio actual: esta home esta renderizada con Bun + EJS.
  • Canales: extension Chrome + PWA instalable.
  • IA: MCP Server y Web Skill para search/lookup/extract.

Menos complejidad operativa, mas velocidad de producto.
